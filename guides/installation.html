Installation Guide

This guide provides detailed instructions for installing and setting up the OmniDragon development environment for different platforms.

Prerequisitesâ€‹

Before installing OmniDragon, ensure you have the following prerequisites:

  • Node.js - v16.x or later
  • npm - v8.x or later
  • Git - Latest version
  • Solidity - Knowledge of Solidity v0.8.x
  • Hardhat - Basic familiarity with Hardhat

Installation Processâ€‹

Step 1: Clone the Repositoryâ€‹

git clone https://github.com/omnidragon/omnidragon.git
cd omnidragon

Step 2: Install Dependenciesâ€‹

npm install

Step 3: Configure Environmentâ€‹

Create your environment configuration:

cp .env.example .env

Edit the .env file with your settings:

# Network RPC URLs
MAINNET_RPC_URL=https://mainnet.infura.io/v3/YOUR_API_KEY
ARBITRUM_RPC_URL=https://arbitrum-mainnet.infura.io/v3/YOUR_API_KEY
BSC_RPC_URL=https://bsc-dataseed.binance.org

# Private keys (NEVER share these)
DEPLOYER_PRIVATE_KEY=your_private_key_here

# API Keys
ETHERSCAN_API_KEY=your_etherscan_api_key
ARBISCAN_API_KEY=your_arbiscan_api_key
BSCSCAN_API_KEY=your_bscscan_api_key

# Contract configuration
INITIAL_SUPPLY=1000000000000000000000000000
FEE_DECIMALS=2

Step 4: Compile Contractsâ€‹

npx hardhat compile

Step 5: Run Testsâ€‹

npx hardhat test

Platform-Specific Instructionsâ€‹

Linux (Ubuntu/Debian)â€‹

Additional packages you might need:

sudo apt update
sudo apt install build-essential python3 python3-dev

macOSâ€‹

Using Homebrew:

brew install node
brew install git

Windowsâ€‹

  1. Install Node.js
  2. Install Git for Windows
  3. Use Windows Terminal or PowerShell

Docker Installationâ€‹

For containerized development:

# Build the image
docker build -t omnidragon .

# Run development container
docker run -it --rm -v $(pwd):/app omnidragon bash

# Inside container
npm install
npx hardhat compile

Troubleshootingâ€‹

Common Issuesâ€‹

"Cannot find module..."â€‹

rm -rf node_modules
npm cache clean --force
npm install

Compilation Errorsâ€‹

npx hardhat clean
npx hardhat compile

Network Connection Issuesâ€‹

Check your .env file for correct RPC URLs and API keys.
